Simple post image gallery

== Description ==

This is simple plugin for add gallery for any post such as product gallery

daneshjooyar post gallery features:

1. simple add images one by one and all in one for each post
2. simple usage in public just by shortcode anywhere in post that you want


== Installation ==

1. Copy daneshjooyar-post-gallery into wp-content/plugins/ directory
2. Activate it from plugins admin menu
3. now you can have image gallery for each post
